YACUSOL - APP CLIENTE PWA - SUS316 + MANTENIMIENTO

NOVEDADES:
- Se conserva SUS 316 en textos y tarjetas.
- Se agrega botón inferior: Mantenimiento.
- El cliente puede ingresar:
  * Nombre
  * Celular
  * Dirección
  * Distrito / ciudad
  * Número de tubos
  * Problema principal
  * Comentario adicional
- Botón para enviar solicitud de mantenimiento por WhatsApp al 942 899 919.

SUBIR A GITHUB:
1. Descomprimir este ZIP.
2. Entrar al repositorio Yacusol-app.
3. Add file > Upload files.
4. Subir TODOS los archivos descomprimidos.
5. Aceptar reemplazar archivos existentes.
6. Commit changes.
7. Esperar 1 a 2 minutos.
8. Abrir la app con Ctrl + F5 o con ?v=mantenimiento.
